# StockTradingApp

A full-stack stock trading web application with:
- **User dashboard** for positions, holdings, orders, and watchlist
- **Trading actions** (buy/sell) wired to backend models
- **React-based landing page** with marketing content

---

## Features

- **Dashboard UI**: Positions, Holdings, Orders, Funds, Summary, Watchlist
- **Trading flows**: Buy/Sell actions routed through backend APIs
- **Reusable charts & components** for portfolio visualization
- **Responsive landing pages** (pricing, about, team, support)


---

## Tech Stack

- **Frontend (React)**: `frontend/`
- **Trading Dashboard (React)**: `dashboard/`
- **Backend (Node.js/JavaScript)**: `backend/`
- **Data models & schemas**:
  - Positions
  - Holdings
  - Orders

---

## Project Structure

- `backend/` — API endpoints, models, and schemas
- `dashboard/` — Trading UI and components
- `frontend/` — Landing page / marketing site
- `README.md` — this documentation

---

## Getting Started

> Assumes you have **Node.js** installed.

### 1) Backend

```bash
cd backend
npm install
npm start
```

### 2) Dashboard

```bash
cd ../dashboard
npm install
npm start
```

### 3) Frontend (Landing page)

```bash
cd ../frontend
npm install
npm start
```

---

## API (Overview)

Backend organizes trading-related data through:
- **PositionsModel** (current holdings/allocations)
- **HoldingsModel** (user assets overview)
- **OrdersModel** (open/filled orders)
